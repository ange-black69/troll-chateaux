# troll-chateaux
Projet de théorie des jeux

-> Le projet se compile facilement comme un projet java 'normal', (il n'y a pas de manager de dépendance comme Maven, Gradle ou encore Ant)

-> Le projet est fourni avec un .jar executable disponible dans le répertoire : \out\artifacts\Troll_Nath_Mike\run.jar


-> le fichier game_result.txt contient la synthèse d'une partie du jeu du troll avec le résultat en terme de victoire et de défaite des deux joueurs.

-> Au lancement du programme, vous avez la possibilité de choisir le nombre de partie à joueur.
